# Поток X — AI Operating System

Статический лендинг для проекта «поток x», выполненный в современном минималистичном стиле. Репозиторий содержит одну веб-страницу `index.html` и набор изображений в папке `screens/` для демонстрации интерфейса.

## О проекте

Этот лендинг представляет концепцию AI operating system с акцентом на:

- элегантный визуальный стиль;
- адаптивную вёрстку для десктопа и мобильных устройств;
- плавные анимации и пользовательский курсор;
- использование графических макетов устройств для демонстрации интерфейса.

## Что внутри

- `index.html` — основная страница лендинга.
- `screens/` — набор изображений для экранных макетов.

## Как развернуть

Просто откройте `index.html` в браузере или используйте любой локальный HTTP-сервер.

Пример запуска через Python:

```bash
cd /Users/yuriyurchenko/projects/lend
python3 -m http.server 8000
```

Затем откройте в браузере:

```
http://localhost:8000
```

## Структура репозитория

- `index.html` — лендинг с оформлением и анимациями.
- `screens/` — графические файлы, используемые на странице.
- `README.md` — документация проекта.

## Особенности

- Отдельная графика для экранов мобильного интерфейса.
- Современные шрифты Google Fonts.
- Прозрачные градиенты, мягкие тени и чистая типографика.
- Нулевой зависимостей и простой статический билд.
